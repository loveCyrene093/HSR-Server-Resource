# 崩坏：星穹铁道服务器数据库

## 文本映射哈希
文本映射的新哈希使用xxhash。您可以找到各种实现方式：

* **JavaScript:** https://github.com/pierrec/js-xxhash
* **Python:** https://github.com/ifduyue/python-xxhash
